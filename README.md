# pc 端全栈文章管理界面

- 运用到的技术栈
  - 前端
    - vue
    - element-ui 框架
    - axios
  - 后端
    - node-express 搭建路由接口
    - mongoDB-mongoose 搭建本地数据库

## 项目运行步骤

1. `npm install` 安装项目中所需的包
2. `node-dev server` 开启后端路由接口以及本地数据库
3. `npm run serve` 即可创建文章并且进行管理
